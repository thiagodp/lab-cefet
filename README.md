# lab-cefet

> Bash script para CMD (Windows) que visa instalar programas necessários ao Laboratórios do Bacharelado em Sistemas de Informação do CEFET/RJ Nova Friburgo

## Como utilizar

> Use por sua conta e risco ⚠️

1. Baixe [install.bat](install.bat) e salve-o em `C:\` (ou outra pasta qualquer).
2. Clique com o botão direito no botão Iniciar (logo do Windows), clique em "Terminal (Administrador)" e confirme para abrir o terminal.
3. Execute `cd C:\` e então `install`.


## Programas instalados

> Todos os programas são instalados em `C:\dev` (ex. `C:\dev\git`).

- Git
- cURL
- PHP
- Composer
- Apache
- MariaDB - inclui MySQL; usuário `root` e senha `root`
- PHPMyAdmin - em `C:\dev\apache\Apache24\htdocs\phpmyadmin`
- NodeJS
- PNPM
- Bun
- Putty
